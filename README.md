# Walking tickets

## Client part setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Server part setup

Create a database and import the file server/walking_tickets_tt_db.sql

```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```
