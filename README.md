# Welbex test task

## Client part setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

Create a database and import the file server/welbex_test_task_db.sql

```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```
